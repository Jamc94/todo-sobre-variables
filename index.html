Hoisting

Se refiere al proceso mediante el cual el intérprete parece mover la declaración de funciones, variables o clases a la parte superior de su alcance, antes de la ejecución del código.

Variable Scopes

El alcance en JavaScript define la visibilidad de variables. Antes de ES6, solo existían alcance global y de función con "var". ES6 introdujo "let" y "const", permitiendo el alcance de bloque. Global se aplica fuera de funciones, función se limita a la función, y bloque se restringe a "{}"

  • Bloque

    Este alcance limita el acceso de variables declaradas en un bloque específico desde el exterior. "let" y "const" facilitan el alcance de bloque. Para acceder, se crea un objeto. Variables con "var" no tienen alcance de bloque.
